# Hello Notesy!

Gather all of your notes in one place, access them from anywhere.

  • Notesy speaks markdown and includes a load of handy shortcuts to make editing a snap

  • Notes are syncronised to a secure server so they’re always safely backed up

  • Available for iPhone and the web so you can edit your notes wherever you are
